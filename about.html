About Me

Hi! I'm Esther, an aspiring designer and frontend developer from Hong Kong, currently doing a Master's degree in Interaction Design in Estonia.

Before pivoting my career to the tech field, I worked in NGOs and government organization as project coordinator and social media manager.

From event planning to web design, my philosophy is the same - be human-centred. I believe that good design not only can improve people's lives, but also can evoke emotions which result in positive user experiences.

When I'm not coding, I'm on the way to cinema / theatre / library / gym room / travel!
